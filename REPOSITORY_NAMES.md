# リポジトリ名のサジェスト

KWU CMS組織のGitHub Pagesまとめサイト用のリポジトリ名候補です。

## 🏆 最推奨

### `kwu-cms.github.io`
- **URL**: `https://kwu-cms.github.io`
- **用途**: 組織のトップレベルページ
- **メリット**: 
  - 最もシンプルで覚えやすい
  - 組織のメインページとして機能
  - SEO的にも有利
- **デメリット**: 組織のメインページとして使うため、他の用途には使えない

## 📋 その他の候補

### プロジェクト関連

| リポジトリ名 | URL | 特徴 |
|---|---|---|
| `projects` | `https://kwu-cms.github.io/projects` | シンプルで分かりやすい |
| `projects-gallery` | `https://kwu-cms.github.io/projects-gallery` | ギャラリー形式を明示 |
| `project-list` | `https://kwu-cms.github.io/project-list` | リスト形式を明示 |

| `repo-listing` | `https://kwu-cms.github.io/repo-listing` | リスト表示を明示 |
| `repo-gallery` | `https://kwu-cms.github.io/repo-gallery` | ギャラリー形式を明示 |

## 💡 命名のポイント

1. **短さ**: 長すぎず、一目で用途がわかること
2. **一貫性**: 他のリポジトリの命名規則に合わせる（例：小文字＋ハイフン区切り）
3. **明確さ**: 何をするものかが名前で伝わること
4. **URLの美しさ**: 覚えやすく、共有しやすいURL

## 🎯 用途別の推奨

- **組織のメインページとして**: `kwu-cms.github.io` ⭐
- **プロジェクト一覧専用**: `projects` または `projects-gallery`
- **リポジトリポータル**: `repo-portal` または `repo-listing`
- **作品展示サイト**: `showcase` または `portfolio`

## 📝 注意事項

- `kwu-cms.github.io` 以外の名前を選ぶ場合、URLは `https://kwu-cms.github.io/{リポジトリ名}` の形式になります
- リポジトリ名を変更する場合は、`script.js` 内のURL生成ロジックも確認してください（現在は自動生成されています）
