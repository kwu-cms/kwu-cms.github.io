# 説明の編集方法ガイド

## 📝 説明を編集する方法

### 手順

1. **編集ボタンを探す**
   - 各リポジトリカードの説明テキストの右上に、小さな鉛筆アイコン（✏️）のボタンがあります
   - マウスを説明の上にホバーすると、ボタンがより見やすくなります

2. **編集ボタンをクリック**
   - 鉛筆アイコンをクリックすると、モーダルウィンドウが開きます

3. **説明を編集**
   - モーダル内のテキストエリアで説明を編集できます
   - 既存の説明が表示されているので、そのまま編集できます

4. **保存**
   - 「保存」ボタンをクリックすると、変更がすぐに画面に反映されます
   - 「キャンセル」ボタンで変更を破棄できます

5. **確認**
   - 保存後、リポジトリカードの説明がすぐに更新されます
   - ページをリロードしても、編集した説明は保持されます

## 💾 保存の仕組み

- 編集した説明は**ブラウザのlocalStorage**に保存されます
- 同じブラウザでアクセスする限り、編集内容は保持されます
- ブラウザのキャッシュをクリアすると、編集内容は消去されます

## 🔄 説明のリセット方法

説明を空にして保存すると、元のGitHubの説明が表示されます。

## 📱 モバイルでの使い方

モバイルデバイスでも同じように編集ボタンをタップして編集できます。

## ⚠️ 注意事項

- 編集内容はローカル（ブラウザ）にのみ保存されます
- 他のデバイスやブラウザでは、編集内容は共有されません
- ブラウザのプライベートモードを使用している場合、ウィンドウを閉じると編集内容が消去される場合があります

## 🎯 使用例

1. **説明を追加する場合**
   - 説明がないリポジトリカードの編集ボタンをクリック
   - テキストエリアに説明を入力
   - 「保存」をクリック

2. **説明を変更する場合**
   - 既存の説明の横にある編集ボタンをクリック
   - テキストエリアで説明を編集
   - 「保存」をクリック

3. **説明を削除する場合**
   - 編集ボタンをクリック
   - テキストエリアの内容をすべて削除
   - 「保存」をクリック（元のGitHubの説明が表示されます）
